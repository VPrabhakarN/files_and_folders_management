## 📁 File & Folder Management System (Python CLI Project)

The **File & Folder Management System** is a command-line based Python project that simulates basic file system operations similar to those performed in an operating system. This project allows users to interact with files and directories directly from the terminal, providing essential functionalities for managing data efficiently.

The system is built using Python's modern **pathlib module** for filesystem path handling and the **shutil module** for directory operations. It demonstrates practical usage of file handling, exception handling, and modular programming concepts.

### 🚀 Key Features

* **Create Folder** – Create new directories in the current working location.
* **View Files & Folders** – Display all files and directories in the current working directory.
* **Rename Folder** – Update folder names safely.
* **Delete Folder** – Remove directories along with their contents.
* **Create File** – Generate new files and add initial content.
* **Read File** – View the contents of existing files.
* **Update File** –

  * Rename a file
  * Append new content
  * Rewrite existing file content
* **Delete File** – Remove files from the system.
